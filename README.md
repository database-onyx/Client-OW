# OneWorld Vacation Website

This is a static website for OneWorld Vacation, built with HTML, Tailwind CSS (via CDN), and Vanilla JavaScript.

## How to Run

Simply open the `index.html` file in any modern web browser.

## Features

-   **Premium Design**: Uses a custom color palette (Emerald Green & Gold) and high-quality typography (Outfit & Playfair Display).
-   **Responsive**: Fully responsive layout for mobile, tablet, and desktop.
-   **Interactive Elements**:
    -   Hover effects on cards and buttons.
    -   Dropdown menus for "India" and "World Tour".
    -   Sticky navbar with scroll effect.
-   **Sections**:
    -   Hero with Search
    -   Features / Icons
    -   Top Destinations
    -   Trip Ideas
    -   Popular Activities
    -   Newsletter CTA
    -   Blog / Inspiration
    -   Testimonials
    -   Footer

## Customization

You can easily customize the colors and fonts in the `<script>` tag within the `<head>` section of `index.html`.
